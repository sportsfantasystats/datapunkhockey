# Data Punk Hockey Repository

This is a public repository of data and resources for hockey analytics enthusiasts. We'll add data, code and tutorials as we complete projects and build out our curriculum. 

You can also check out www.datapunkhockey.com for our newsletters and more content/courseware. 

## Introduction to Hockey Analytics

As a supplement to our introductory series on hockey analytics, we've added three examples here for enthusiasts to check out. 

These examples are described as follows: 

### Level 1 Example

Provides a raw and cleaned version of NHL data sourced from www.hockey-reference.com. It also provides a simple analysis to answer the question of "Who are the Top 5 NHL Teams?" The analytics here would be in line with a hobbyist or beginning analyst.

### Level 2 Example

Provides two separate examples of Power BI dashboards. The dashboards are already completed. One of for the implementation of a Balance metric, and the other is a comparative report for a past NHL event.

### Level 3 Example

Provides an example of providing probabilities for how to predict who will win a game. Uses the Pythagorean Win Probability (aka Pythagorean Expectation) to do this.
